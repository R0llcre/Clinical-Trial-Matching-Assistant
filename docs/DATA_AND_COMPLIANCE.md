Data And Compliance

**数据来源**
- ClinicalTrials.gov Data API
- Synthea 合成患者数据
- 手动输入患者画像

**隐私与边界**
- 禁止处理真实 EHR 或可识别个人健康信息
- 系统仅提供信息检索与条款解释
- UI 与文档必须明确免责声明

**数据使用规范**
- 试验数据仅用于展示与检索
- 保留 raw_json 与 data_timestamp
- 不修改外部数据来源内容

**缓存与限流**
- 使用 data_timestamp 驱动缓存失效
- 外部 API 请求速率控制
- 429/5xx 采用指数退避

**日志与审计**
- 记录请求 ID 与任务 ID
- 不记录患者自由文本原文
- 记录解析失败原因与重试次数

**LLM 使用与降级**
- 解析与摘要均可关闭
- 失败或超预算时自动降级
- 输出必须通过结构化校验与 evidence 对齐
